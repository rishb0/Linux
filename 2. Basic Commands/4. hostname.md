# `hostname` ‚Äî Show or Set System Hostname

Displays or changes the hostname of the system. The hostname is the name that identifies the machine on a network.

---
## Syntax

```
hostname [options] [new-hostname]
```

---
## Basic Usage

Show current hostname:

```bash
hostname
```

Show the full FQDN (Fully Qualified Domain Name):

```bash
hostname -f
```

Show only the domain part:

```bash
hostname -d
```

Show the short hostname (without domain):

```bash
hostname -s
```

Show all IP addresses associated with the hostname:

```bash
hostname -I
```

Show the system's IP address (single):

```bash
hostname -i
```

---

## Changing the Hostname

### Temporary ‚Äî resets after reboot

Set hostname temporarily (takes effect immediately, lost on reboot):

```bash
sudo hostname newhostname
```

### Permanent ‚Äî survives reboot

#### Method 1 ‚Äî using `hostnamectl` (systemd systems ‚Äî recommended)

Set hostname permanently:

```bash
sudo hostnamectl set-hostname newhostname
```

Verify the change:

```bash
hostnamectl
```

#### Method 2 ‚Äî editing the hostname file directly

Open and edit the hostname file:

```bash
sudo nano /etc/hostname
```

Replace the existing name with the new one, save, then reboot or run:

```bash
sudo hostname $(cat /etc/hostname)
```

#### Method 3 ‚Äî updating `/etc/hosts` (always do this alongside Method 1 or 2)

Open the hosts file:

```bash
sudo nano /etc/hosts
```

Find the line with the old hostname and update it:

```
127.0.0.1   localhost
127.0.1.1   newhostname
```

> Updating `/etc/hosts` is important ‚Äî some applications and tools resolve the local hostname through this file. Not updating it can cause warnings, slow logins, or broken sudo.

## `hostnamectl` ‚Äî systemd Hostname Tool

`hostnamectl` manages three types of hostnames:

|Type|Description|
|---|---|
|**Static**|The traditional persistent hostname stored in `/etc/hostname`|
|**Transient**|Temporary hostname set by DHCP or kernel at runtime|
|**Pretty**|A free-form human-readable name (can include spaces, e.g., "Risabh's Kali Box")|

Show full hostname info:

```bash
hostnamectl
```

Set static hostname:

```bash
sudo hostnamectl set-hostname newhostname
```

Set pretty hostname:

```bash
sudo hostnamectl set-hostname "Risabh's Kali Box" --pretty
```

---

## Key Files

|File|Purpose|
|---|---|
|`/etc/hostname`|Stores the persistent static hostname|
|`/etc/hosts`|Local hostname-to-IP mapping; must be updated alongside hostname changes|

---

## Options Summary

|Option|Meaning|
|---|---|
|(none)|Display current short hostname|
|`-f`|Display FQDN (Fully Qualified Domain Name)|
|`-d`|Display domain name only|
|`-s`|Display short hostname only|
|`-I`|Display all IP addresses of the host|
|`-i`|Display IP address for the hostname|
