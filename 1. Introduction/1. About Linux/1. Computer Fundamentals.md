# Computer Fundamentals

## 1. What is a Computer?

A computer is an electronic device that accepts raw data as input, processes that data according to a set of instructions (called a program), stores the data and results, and produces meaningful output. The key characteristic that separates a computer from a simple calculator is that a computer is **programmable** — you can change what it does by changing the software running on it, without changing the hardware. Computers operate at extremely high speeds, measured in millions or billions of operations per second, and they work with data represented in **binary** — a number system using only 0s and 1s, because electronic circuits have two states: off (0) and on (1).

## 2. Basic Components of a Computer

### Hardware — The Physical Parts

**CPU (Central Processing Unit)** is the brain of the computer. It fetches instructions from memory, decodes what each instruction means, and executes it. The CPU contains an **ALU (Arithmetic Logic Unit)** that performs math and logical comparisons, **Registers** which are tiny ultra-fast storage locations inside the CPU itself, and a **Control Unit** that coordinates everything. CPU speed is measured in GHz (gigahertz — billions of cycles per second). Modern CPUs have multiple **cores**, meaning multiple independent processing units on one chip, allowing true parallel execution of tasks.

**RAM (Random Access Memory)** is the computer's short-term, working memory. When you open a program, it is copied from the storage drive into RAM because RAM is vastly faster than any drive. RAM is **volatile** — it loses all its data the moment power is cut. RAM capacity is measured in GB. The more RAM you have, the more programs you can run simultaneously without the system slowing down. From a security perspective, RAM is critical because sensitive data like encryption keys, passwords, and plaintext of decrypted files lives in RAM while in use — this is what cold boot attacks exploit.

**Storage (HDD / SSD)** is long-term, non-volatile memory — data stays here even when power is off. **HDD (Hard Disk Drive)** uses spinning magnetic platters with a read/write head; they are slow but cheap and high capacity. **SSD (Solid State Drive)** uses flash memory chips with no moving parts; they are significantly faster, more durable, and more expensive per GB. Storage is where the OS, programs, and all your files permanently reside.

**Motherboard** is the main circuit board that physically connects and allows communication between the CPU, RAM, storage, GPU, and all other components. It contains the **chipset** which manages data flow between components, **BIOS/UEFI firmware** on a dedicated chip, expansion slots (PCIe), RAM slots, and connectors for all peripherals.

**GPU (Graphics Processing Unit)** is a specialized processor originally designed to render images. It contains thousands of smaller cores optimized for parallel tasks. In cybersecurity, GPUs are heavily used for **password cracking** — tools like Hashcat use the GPU's massive parallelism to test billions of password guesses per second against captured password hashes.

**Power Supply Unit (PSU)** converts AC power from the wall outlet into the DC voltages (3.3V, 5V, 12V) that computer components need.

**Input Devices** are anything that sends data into the computer — keyboard, mouse, microphone, webcam, biometric scanners. In security contexts, compromised input devices (like a hardware keylogger plugged between a keyboard and PC) are a real physical attack vector.

**Output Devices** are anything that delivers processed data out to the user — monitor, printer, speakers.

### Software — The Instructions

**System Software** is the layer of software that manages the hardware and provides a platform for other software to run on. The most important system software is the **Operating System (OS)** — examples are Windows, Linux, and macOS. The OS manages the CPU (deciding which program gets processor time), manages RAM (deciding which program gets which memory space), handles storage (filesystem management), controls security (user permissions, access control), and provides an interface between application software and hardware. Without an OS, each application would have to directly manage all hardware itself, which would be impossibly complex.

**Application Software** are programs built on top of the OS that serve specific user purposes — a web browser, a text editor, a game, a hacking tool like Nmap or Wireshark. Applications make system calls to the OS when they need hardware resources; they cannot access hardware directly in modern systems (this is a security boundary called **ring protection**, covered in OS topics).

## 3. How a Computer Works — The Processing Cycle

Every operation a computer performs follows this cycle, also called the **IPO (Input → Process → Output)** model, extended with storage:

**Input** — Data enters the computer through an input device or from storage. For example, you type a URL into a browser.

**Processing** — The CPU executes program instructions on the input data. The browser's code processes your URL, makes a network request, receives HTML data, parses it, and renders it.

**Storage** — Data is saved either temporarily in RAM (while in use) or permanently on a drive. Cached web pages, for instance, are stored on your SSD for faster future access.

**Output** — The result is presented to the user — the rendered webpage appears on your monitor.

At the hardware level, the CPU continuously runs the **Fetch-Decode-Execute cycle**: it fetches the next instruction from RAM (using the **Program Counter** register to know the address), decodes what that instruction means, executes it, and moves to the next instruction. This happens billions of times per second.

## 4. Types of Computers

**Desktop Computers** are fixed-location systems with separate components — tower (containing CPU, RAM, storage), monitor, keyboard, and mouse. They offer better performance per rupee and are easier to upgrade and repair than laptops. Common in offices, homes, and for gaming or development work.

**Laptop Computers** are portable, battery-powered computers with all components integrated into a single unit. They sacrifice some performance and upgradeability for portability. The battery introduces the concept of power management — the CPU and other components throttle their speed to preserve battery.

**Servers** are high-reliability computers designed to run continuously, provide services, and handle requests from multiple other computers (clients) simultaneously. Servers typically have more RAM, more storage, redundant power supplies, and error-correcting memory. They run server operating systems like Windows Server or Linux. In cybersecurity, servers are primary targets because they hold the most valuable data and are exposed to networks. A web server hosts websites; a database server stores data; a DNS server resolves domain names; a file server stores shared files.

**Supercomputers** are the most powerful computers in existence, built from thousands of processors working in parallel. They are used for weather forecasting, nuclear simulations, AI model training, drug discovery, and cryptographic research. Their processing power is measured in FLOPS (Floating Point Operations Per Second) — modern supercomputers operate in the petaflop to exaflop range.

**Embedded Systems** are dedicated computers built into other devices to control specific functions. They have a fixed program stored in ROM and are not general-purpose. Examples include the computer inside a car's ABS braking system, inside a router, inside an ATM, inside a smart TV, or inside an industrial control system (ICS/SCADA). Embedded systems are a growing cybersecurity concern — they often run outdated software, have no display or interface for updates, and are forgotten about yet connected to networks (this is the core of the IoT security problem).

**Mobile Devices (Smartphones/Tablets)** are essentially powerful computers in a small form factor, running mobile operating systems like Android (Linux-based) or iOS. They contain CPUs, RAM, flash storage, and multiple radios (cellular, WiFi, Bluetooth, GPS). In ethical hacking, mobile devices are a major attack surface.

## 5. Basic Computer Security — Why Each Point Actually Matters

**Strong Passwords** protect accounts because the most common way systems are compromised is through authentication — attackers try to log in as a legitimate user. A weak password (like "123456" or "password") can be guessed or cracked in seconds using dictionary attacks or brute force. A strong password is long (12+ characters), uses a mix of uppercase, lowercase, numbers, and symbols, and is unique per account. **Password managers** generate and store these. From a hacking perspective, tools like Hydra and Medusa automate password guessing attacks against login systems.

**Software Updates (Patching)** fix known vulnerabilities. When a security flaw is discovered in software, the developer releases a patch. The window of time between a vulnerability being publicly disclosed and users patching their systems is when attackers are most active — these are called **n-day exploits**. If attackers find a vulnerability before the developer does, it is called a **zero-day (0-day)** — the most dangerous category because no patch exists yet. Unpatched systems are the single biggest entry point in real-world attacks.

**Antivirus / Antimalware** software detects, blocks, and removes malicious programs. It works using **signature-based detection** (comparing files against a database of known malware signatures) and **heuristic/behavioral detection** (flagging programs that behave suspiciously, even if not in the database). Modern endpoint security products (called **EDR — Endpoint Detection and Response**) are far more advanced than traditional antivirus. Importantly, antivirus is not foolproof — skilled attackers specifically craft malware to evade detection, which is a major topic in ethical hacking (AV evasion).

**Avoiding Suspicious Links** protects against **phishing** — the most common social engineering attack, where an attacker sends a fake link that looks legitimate (like a fake bank login page) to steal credentials, or a link that downloads malware. Before clicking any link, you should verify the actual domain in the URL (hovering over it reveals the real destination), check for HTTPS, and be especially suspicious of unexpected emails asking you to take urgent action — urgency is a manipulation technique.

**Firewall** (worth adding here) is software or hardware that monitors and controls incoming and outgoing network traffic based on rules. It acts as a barrier between your computer and the network, blocking unauthorized connections. Windows has a built-in software firewall; routers contain hardware firewalls. In ethical hacking, firewall enumeration and evasion are standard skills.

**Encryption** protects stored data (like BitLocker encrypting your drive) and data in transit (like HTTPS encrypting your web traffic) so that even if an attacker intercepts or steals your data, they cannot read it without the key.
