# Linux Distribution Family Tree

> Indentation = derived from parent above it. Distros at the same level under the same parent share that parent as their direct base.

---

    - Kali Linux _(direct Debian base)_
    - Parrot Security OS _(direct Debian base)_
    - MX Linux _(direct Debian Stable base)_
    - Deepin _(direct Debian base, since v20)_
    - SparkyLinux _(direct Debian base)_
    - Raspberry Pi OS _(direct Debian base; formerly Raspbian)_
    - Tails _(direct Debian base)_
    - Knoppix _(direct Debian base; live distro)_
    - Slax _(direct Debian base; live distro)_
    - **Ubuntu** _(direct Debian base)_
        - Linux Mint _(Ubuntu-based; also has LMDE edition which is direct Debian)_
        - Elementary OS
        - Zorin OS
        - Edubuntu
        - Ubuntu Server _(Ubuntu flavour)_
        - Ubuntu Studio _(Ubuntu flavour)_
        - Lubuntu _(Ubuntu flavour — LXQt desktop)_
        - Xubuntu _(Ubuntu flavour — XFCE desktop)_

        - Ubuntu MATE _(Ubuntu flavour — MATE desktop)_
        - Ubuntu Touch _(maintained by UBports)_
        - Rescuezilla
        - **Linux Mint** → **LMDE** _(Linux Mint Debian Edition — direct Debian, not Ubuntu)_

## Red Hat-based

- **Fedora** _(upstream / testing ground — new features go here first)_
    - **Red Hat Enterprise Linux (RHEL)** _(built from stabilised Fedora code)_
        - CentOS Stream _(rolling upstream preview of RHEL minor releases)_
        - AlmaLinux _(free, binary-compatible RHEL rebuild)_
        - Rocky Linux _(free, binary-compatible RHEL rebuild)_
        - Oracle Linux _(RHEL rebuild with Oracle's UEK kernel)_
        - Scientific Linux _(discontinued 2021; was a RHEL rebuild)_

> **Note:** Fedora is _upstream_ of RHEL — it is not derived from RHEL. The flow is: **Fedora → RHEL → (AlmaLinux / Rocky / Oracle / CentOS Stream)**. CentOS Stream sits between Fedora and RHEL as a continuous preview of the next RHEL minor version.

## Arch-based

    - Manjaro
    - EndeavourOS
    - ArcoLinux
    - SteamOS 3.x _(Valve; used on Steam Deck)_

---

## SUSE-based

- **SUSE Linux Enterprise Server (SLES)** _(commercial enterprise base)_
    - **openSUSE Leap** _(community distro built from SLES + openSUSE packages)_
    - **openSUSE Tumbleweed** _(independent rolling release; shares openSUSE infrastructure)_
    - **GeckoLinux** _(openSUSE respin with better out-of-the-box experience)_

> **Note:** openSUSE Tumbleweed is not derived from Leap or SLES — it is a parallel independent rolling branch under the openSUSE project.

---

## Gentoo-based

- **Gentoo**
    - Funtoo _(fork by Gentoo's original creator Daniel Robbins)_
    - Calculate Linux _(Russian-developed; desktop and server focus)_
    - Chrome OS _(Google; uses Gentoo's Portage build system)_
    - Sabayon Linux _(discontinued 2021; community moved to Macaroni OS)_

## Slackware-based

    - Zenwalk
    - VectorLinux

## Independent Distributions

_(Not derived from any major family above)_

- NixOS _(declarative config; Nix package manager)_
- Solus _(desktop-first; eopkg package manager)_
- Clear Linux _(Intel-developed; performance-optimised)_
- Alpine Linux _(musl libc + BusyBox; minimal; default Docker base image)_
    - postmarketOS _(Alpine-based; for smartphones)_
- Puppy Linux _(ultra-lightweight; runs from RAM)_
- Qubes OS _(Xen-based VM isolation per app)_
- Tiny Core Linux _(~11 MB; boots into RAM)_
- LFS — Linux From Scratch _(build your own Linux; educational)_
- CRUX _(minimalist; ports-based)_
- Source Mage _(source-based; Sorcery package system)_
