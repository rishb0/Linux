### Tpes of ROM:

- **[Mask ROM (MROM)](https://www.google.com/search?sca_esv=3587ac9b8d8afde8&sxsrf=ANbL-n6wCFQY05yFJkLI1bjLW9GVuOm3lA%3A1771777868533&q=Mask+ROM+%28MROM%29&source=lnms&fbs=ADc_l-byipRaccqV0jmfPhi1DgzPkIbk0HG8UBQNbHd4S9AjVkuc0t6t4dDUPoihiWakL0i6q2BZxOD3Y5tb96YpHp1ExkuspSSKmqXubPqldj9nEKNrUcLzLKrjPAW4FxjEoLxr3EWbm8KDvabwQovHNrmC8gyoPTmdZ_m_xO5vRhsduL4O_K-IJmIWg-sd82JB09Jfd7REO4iocY-9E3vr9BomJTt3CeeXwlCGMLegY3B8xfYZmt4&sa=X&ved=2ahUKEwiv0O6Aw-2SAxWXRWwGHbhlEo4QgK4QegYIAQgAEAo&biw=1707&bih=938&dpr=1.13):** The earliest, most economical, and obsolete type, where data is hard-wired into the chip during manufacturing. It cannot be changed.
- **[Programmable Read-Only Memory (PROM)](https://www.google.com/search?sca_esv=3587ac9b8d8afde8&sxsrf=ANbL-n6wCFQY05yFJkLI1bjLW9GVuOm3lA%3A1771777868533&q=Programmable+Read-Only+Memory+%28PROM%29&source=lnms&fbs=ADc_l-byipRaccqV0jmfPhi1DgzPkIbk0HG8UBQNbHd4S9AjVkuc0t6t4dDUPoihiWakL0i6q2BZxOD3Y5tb96YpHp1ExkuspSSKmqXubPqldj9nEKNrUcLzLKrjPAW4FxjEoLxr3EWbm8KDvabwQovHNrmC8gyoPTmdZ_m_xO5vRhsduL4O_K-IJmIWg-sd82JB09Jfd7REO4iocY-9E3vr9BomJTt3CeeXwlCGMLegY3B8xfYZmt4&sa=X&ved=2ahUKEwiv0O6Aw-2SAxWXRWwGHbhlEo4QgK4QegYIAQgAEAw&biw=1707&bih=938&dpr=1.13):** A blank memory chip that can be programmed once by the user using a special tool (PROM burner)
    
    . Once programmed, it cannot be erased or modified.
    

- **[Erasable Programmable Read-Only Memory (EPROM)](https://www.google.com/search?sca_esv=3587ac9b8d8afde8&sxsrf=ANbL-n6wCFQY05yFJkLI1bjLW9GVuOm3lA%3A1771777868533&q=Erasable+Programmable+Read-Only+Memory+%28EPROM%29&source=lnms&fbs=ADc_l-byipRaccqV0jmfPhi1DgzPkIbk0HG8UBQNbHd4S9AjVkuc0t6t4dDUPoihiWakL0i6q2BZxOD3Y5tb96YpHp1ExkuspSSKmqXubPqldj9nEKNrUcLzLKrjPAW4FxjEoLxr3EWbm8KDvabwQovHNrmC8gyoPTmdZ_m_xO5vRhsduL4O_K-IJmIWg-sd82JB09Jfd7REO4iocY-9E3vr9BomJTt3CeeXwlCGMLegY3B8xfYZmt4&sa=X&ved=2ahUKEwiv0O6Aw-2SAxWXRWwGHbhlEo4QgK4QegYIAQgCEAE&biw=1707&bih=938&dpr=1.13):** Can be erased and rewritten multiple times by exposing the chip to ultraviolet (UV) light for a specific duration.
- **[Electrically Erasable Programmable Read-Only Memory (EEPROM)](https://www.google.com/search?sca_esv=3587ac9b8d8afde8&sxsrf=ANbL-n6wCFQY05yFJkLI1bjLW9GVuOm3lA%3A1771777868533&q=Electrically+Erasable+Programmable+Read-Only+Memory+%28EEPROM%29&source=lnms&fbs=ADc_l-byipRaccqV0jmfPhi1DgzPkIbk0HG8UBQNbHd4S9AjVkuc0t6t4dDUPoihiWakL0i6q2BZxOD3Y5tb96YpHp1ExkuspSSKmqXubPqldj9nEKNrUcLzLKrjPAW4FxjEoLxr3EWbm8KDvabwQovHNrmC8gyoPTmdZ_m_xO5vRhsduL4O_K-IJmIWg-sd82JB09Jfd7REO4iocY-9E3vr9BomJTt3CeeXwlCGMLegY3B8xfYZmt4&sa=X&ved=2ahUKEwiv0O6Aw-2SAxWXRWwGHbhlEo4QgK4QegYIAQgCEAM&biw=1707&bih=938&dpr=1.13):** Can be erased and rewritten electrically, making it much more flexible than EPROM as it does not require UV light or removal from the device.
- **[Flash Memory](https://www.google.com/search?sca_esv=3587ac9b8d8afde8&sxsrf=ANbL-n6wCFQY05yFJkLI1bjLW9GVuOm3lA%3A1771777868533&q=Flash+Memory&source=lnms&fbs=ADc_l-byipRaccqV0jmfPhi1DgzPkIbk0HG8UBQNbHd4S9AjVkuc0t6t4dDUPoihiWakL0i6q2BZxOD3Y5tb96YpHp1ExkuspSSKmqXubPqldj9nEKNrUcLzLKrjPAW4FxjEoLxr3EWbm8KDvabwQovHNrmC8gyoPTmdZ_m_xO5vRhsduL4O_K-IJmIWg-sd82JB09Jfd7REO4iocY-9E3vr9BomJTt3CeeXwlCGMLegY3B8xfYZmt4&sa=X&ved=2ahUKEwiv0O6Aw-2SAxWXRWwGHbhlEo4QgK4QegYIAQgCEAU&biw=1707&bih=938&dpr=1.13):** A modern, faster form of EEPROM that allows data to be erased and reprogrammed in blocks. It is widely used in USB drives and solid-state drives (SSDs).
